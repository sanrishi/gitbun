# Gitbun VSCode Extension  Quickstart
## Running the Extension
- Press `F5` to open a new VS Code window with the extension loaded
- Stage some changes, then run `Gitbun: Generate Commit Message` from the Command Palette
- Or use the keybinding `Ctrl+Shift+G G`
## Debugging
- Set breakpoints in `src/extension.ts` or `src/gitbun.ts`
- View logs in Output > Gitbun channel
